# JPeMANY Soluções Digitais

Site oficial da JPeMANY Soluções Digitais, desenvolvido com foco em performance, design moderno e experiência do usuário.

O projeto apresenta os serviços da empresa, portfólio, informações institucionais e soluções digitais voltadas para empresas e profissionais.

## 🚀 Tecnologias Utilizadas

- TypeScript
- Next.js
- React
- Tailwind CSS
- Responsive Design

## ✨ Funcionalidades

- Página institucional moderna
- Layout responsivo
- Sessões de serviços
- Portfólio de projetos
- Navegação otimizada
- Estrutura escalável

## 🎯 Objetivo

Criar uma presença digital profissional para a empresa, transmitindo criatividade, modernidade e confiabilidade.
